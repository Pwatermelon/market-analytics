# Быстрый старт

## Шаг 1: Скачивание моделей

```bash
pip install -r requirements.txt
python download_models.py
```

Это займет некоторое время (скачается ~1 GB моделей).

## Шаг 2: Запуск приложения

```bash
docker-compose up --build
```

При первом запуске Docker будет собирать образы и скачивать зависимости. Это может занять 10-15 минут.

## Шаг 3: Открытие приложения

После запуска откройте в браузере:
- **Frontend**: http://localhost:3000
- **API Gateway**: http://localhost:8000
- **API Docs**: http://localhost:8000/docs

## Шаг 4: Регистрация и использование

1. Зарегистрируйтесь на http://localhost:3000/register
2. Добавьте товар для отслеживания (укажите URL с маркетплейса)
3. Нажмите "Парсить отзывы" для сбора отзывов
4. Нажмите "Анализировать" для анализа тональности
5. Просмотрите графики и суммаризацию на странице товара

## Поддерживаемые маркетплейсы

- **Wildberries** - полная поддержка
- **Ozon** - в разработке
- **Яндекс.Маркет** - в разработке

## Troubleshooting

### Модели не загружаются
- Убедитесь, что папка `models/` содержит подпапки `sentiment/` и `summarizer/`
- Проверьте логи: `docker-compose logs analyzer-service`

### Ошибки при парсинге
- Убедитесь, что URL товара корректный
- Проверьте логи: `docker-compose logs parser-service`

### Проблемы с базой данных
- Остановите контейнеры: `docker-compose down`
- Удалите volume: `docker-compose down -v`
- Запустите заново: `docker-compose up --build`
