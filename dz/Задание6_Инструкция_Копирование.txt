ЗАДАНИЕ 6: ФОРМЫ ПОЛЬЗОВАТЕЛЯ - ИНСТРУКЦИЯ ПО КОПИРОВАНИЮ
═══════════════════════════════════════════════════════════════

ШАГ 1: Создайте форму пользователя
───────────────────────────────────
1. Откройте Excel
2. Нажмите Alt+F11 (открыть редактор VBA)
3. Вставка → UserForm
4. Создайте форму с элементами (см. ниже)

ШАГ 2: Добавьте элементы на форму
───────────────────────────────────
Используйте панель инструментов для добавления:

┌─────────────────────────────────────────────┐
│  Вычисление параметров ромба               │
│                                             │
│  Диагональ 1:  [TextBox1]                   │
│  Диагональ 2:  [TextBox2]                   │
│                                             │
│  Площадь:      [TextBox3]                   │
│  Угол α:       [TextBox4]                   │
│  Угол β:       [TextBox5]                   │
│  Сторона:      [TextBox6]                   │
│  Периметр:     [TextBox7]                   │
│                                             │
│  [Вычислить]  [Очистить]  [Выход]           │
└─────────────────────────────────────────────┘

Элементы:
- Label1: "Диагональ 1:"
- TextBox1: для ввода d1
- Label2: "Диагональ 2:"
- TextBox2: для ввода d2
- Label3: "Площадь:"
- TextBox3: для вывода площади
- Label4: "Угол α:"
- TextBox4: для вывода угла α
- Label5: "Угол β:"
- TextBox5: для вывода угла β
- Label6: "Сторона:"
- TextBox6: для вывода стороны
- Label7: "Периметр:"
- TextBox7: для вывода периметра
- CommandButton1: "Вычислить"
- CommandButton2: "Очистить"
- CommandButton3: "Выход"

ШАГ 3: Добавьте код
─────────────────────
1. Двойной клик по форме (или CommandButton1)
2. Скопируйте ВЕСЬ код из файла "Задание6_ФормыПользователя.bas"
3. Вставьте в окно кода формы
4. Сохраните файл как .xlsm

ШАГ 4: Запустите форму
───────────────────────
1. Нажмите Alt+F8
2. Выберите макрос "PokazatFormuRomb"
3. Нажмите "Выполнить"
4. Или создайте кнопку на листе и привяжите этот макрос
