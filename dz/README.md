## Формулы для ромба:

- Площадь: S = (d1 × d2) / 2, где d1 и d2 - диагонали
- Угол α: α = 2 × arctan(d2/d1)
- Угол β: β = 180° - α
- Сторона: a = √((d1/2)² + (d2/2)²)
- Периметр: P = 4a
