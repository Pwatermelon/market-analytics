КОНТРОЛЬНОЕ ЗАДАНИЕ 5
Вариант 7: Вычисление площади и углов ромба по его диагоналям
Реализация с элементами управления

ИНСТРУКЦИЯ ДЛЯ EXCEL:

1. Подготовка листа:
   - Создайте таблицу с полями для ввода и вывода
   - A1: "Диагональ 1"
   - B1: (пустая ячейка для ввода)
   - A2: "Диагональ 2"
   - B2: (пустая ячейка для ввода)
   - A3-A7: Параметры (Площадь, Угол α, Угол β, Сторона, Периметр)
   - B3-B7: (ячейки для результатов)

2. Добавление элементов управления (Форма):
   - Разработчик → Вставить → Элементы управления формы
   - Добавьте:
     * Поле ввода (для d1) → привяжите к ячейке B1
     * Поле ввода (для d2) → привяжите к ячейке B2
     * Кнопка "Вычислить" → привяжите макрос из Задания 3

3. Добавление элементов управления (ActiveX):
   - Разработчик → Вставить → Элементы управления ActiveX
   - Добавьте:
     * TextBox1 (для d1) → свойство LinkedCell = B1
     * TextBox2 (для d2) → свойство LinkedCell = B2
     * CommandButton1 → код:
       Private Sub CommandButton1_Click()
           Call ВычислитьПараметрыРомба
       End Sub
     * Label для отображения результатов

4. Настройка макроса (см. Задание3_Ромб_Макрос.bas)

ИНСТРУКЦИЯ ДЛЯ WORD:

1. Создайте таблицу с расчетами (см. Задание 1)

2. Добавьте элементы управления:
   - Разработчик → Элементы управления
   - Добавьте:
     * Поле формы (для d1)
     * Поле формы (для d2)
     * Кнопка "Вычислить"
     * Поля для результатов

3. Используйте поля формы с расчетами:
   - Вставка → Поле → Формула
   - Или используйте закладки и поля REF

4. Альтернативно: вставьте объект Excel с элементами управления

ПРИМЕР МАКРОСА ДЛЯ КНОПКИ:

Sub КнопкаВычислить()
    Dim d1 As Double, d2 As Double
    d1 = Range("B1").Value
    d2 = Range("B2").Value
    
    Range("B3").Value = (d1 * d2) / 2
    Range("B4").Value = 2 * Atn(d2 / d1) * 180 / Application.WorksheetFunction.Pi()
    Range("B5").Value = 180 - Range("B4").Value
    Range("B6").Value = Sqr((d1 / 2) ^ 2 + (d2 / 2) ^ 2)
    Range("B7").Value = 4 * Range("B6").Value
End Sub
