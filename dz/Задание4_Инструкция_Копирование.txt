ЗАДАНИЕ 4: КОРНИ ФУНКЦИИ - ИНСТРУКЦИЯ ПО КОПИРОВАНИЮ

ШАГ 1: Создайте таблицу параметров
───────────────────────────────────
Скопируйте следующее в Excel (начиная с ячейки A1):

┌──────────────────────────────┬──────────┐
│ Параметр                     │ Значение │
├──────────────────────────────┼──────────┤
│ x1 (начало интервала)        │ 0.1      │
│ x2 (конец интервала)          │ 10       │
│ A_min                        │ -5       │
│ A_max                        │ 5        │
│ A_step                       │ 0.5      │
│ B_min                        │ -5       │
│ B_max                        │ 5        │
│ B_step                       │ 0.5      │
└──────────────────────────────┴──────────┘

ИЛИ откройте файл "Задание4_Корни_Данные.csv"

ШАГ 2: Добавьте макрос
───────────────────────
1. Нажмите Alt+F11 (открыть редактор VBA)
2. Вставка → Модуль
3. Скопируйте код из файла "Задание4_КорниФункции.bas"
4. Сохраните файл как .xlsm

ШАГ 3: Запустите макрос
────────────────────────
1. Нажмите Alt+F8
2. Выберите макрос "NaytiKoefitsienty"
3. Нажмите "Выполнить"
4. Результаты появятся в столбцах D-F
