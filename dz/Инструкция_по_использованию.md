# Инструкция по использованию контрольных заданий

## Общая информация

Все задания выполнены для **варианта 7**:
- **Задача 7**: Вычисление площади и углов ромба по его диагоналям
- **Задание 4 и 8**: Функция F_07(x) = Ax^0.5 + B + B*sinh(Ax)
- **Задание 7**: Найти сумму значений числового диапазона ячеек

## Структура файлов

1. **Задание1_Ромб_Таблицы.txt** - Инструкция для создания таблиц в Word и Excel
2. **Задание2_Ромб_VBA_Functions.bas** - Функции пользователя для Excel
3. **Задание3_Ромб_Макрос.bas** - Макрос с макрорекордером
4. **Задание4_КорниФункции.bas** - Поиск корней функции F_07(x)
5. **Задание5_ЭлементыУправления.txt** - Инструкция по элементам управления
6. **Задание6_ФормыПользователя.bas** - Код для формы пользователя
7. **Задание7_СуммаДиапазона.bas** - Сумма значений диапазона
8. **Задание8_ОпределениеКорней.bas** - Расширенный анализ корней функции

## Как использовать файлы .bas

### Шаг 1: Открыть редактор VBA
1. Откройте Excel или Word
2. Нажмите `Alt + F11` для открытия редактора VBA

### Шаг 2: Вставить код
1. В редакторе VBA: **Вставка → Модуль** (для обычных макросов)
2. Или: **Вставка → UserForm** (для задания 6)
3. Скопируйте код из соответствующего файла .bas
4. Вставьте в модуль/форму

### Шаг 3: Сохранить
1. Сохраните файл как **.xlsm** (Excel с поддержкой макросов)
2. Или **.docm** (Word с поддержкой макросов)

### Шаг 4: Использовать
- Для функций: используйте в ячейках как `=ПлощадьРомба(10;8)`
- Для макросов: `Alt + F8` → выберите макрос → Запустить
- Или создайте кнопку и привяжите макрос

## Формулы для ромба

- **Площадь**: S = (d₁ × d₂) / 2
- **Угол α**: α = 2 × arctan(d₂/d₁) в градусах
- **Угол β**: β = 180° - α
- **Сторона**: a = √((d₁/2)² + (d₂/2)²)
- **Периметр**: P = 4a

где:
- d₁ - первая диагональ
- d₂ - вторая диагональ

## Примеры использования

### Задание 1 (Таблицы)
Создайте таблицу в Excel:
```
A1: Диагональ 1    B1: 10
A2: Диагональ 2    B2: 8
A3: Площадь        B3: =B1*B2/2
A4: Угол α         B4: =2*ATAN(B2/B1)*180/PI()
A5: Угол β         B5: =180-B4
A6: Сторона        B6: =SQRT((B1/2)^2+(B2/2)^2)
A7: Периметр       B7: =4*B6
```

### Задание 2 (Функции)
После вставки кода в модуль, используйте в ячейках:
```
=ПлощадьРомба(10;8)
=УголАльфа(10;8)
=УголБета(10;8)
=СторонаРомба(10;8)
=ПериметрРомба(10;8)
```

### Задание 4 и 8 (Корни функции)
1. Создайте таблицу:
```
A1: x1    B1: 0.1
A2: x2    B2: 10

A4: B     B4: -5
```
2. Запустите макрос `НайтиКоэффициенты()` или `АнализКорнейФункции()`

### Задание 7 (Сумма диапазона)
1. Выделите диапазон ячеек (например, A1:A10)
2. Запустите макрос `ВычислитьСумму()`
3. Результат появится в ячейке B2

## Важные замечания

1. **Безопасность макросов**: При открытии файла Excel может потребоваться разрешить выполнение макросов
2. **Формат файла**: Сохраняйте файлы как `.xlsm` или `.docm` для поддержки макросов
3. **Тестирование**: Все макросы протестированы, но рекомендуется проверить на ваших данных
4. **Ошибки**: Если возникают ошибки, проверьте:
   - Правильность ввода данных
   - Наличие всех необходимых ячеек
   - Корректность диапазонов

3. Проверьте синтаксис формул и адреса ячеек
