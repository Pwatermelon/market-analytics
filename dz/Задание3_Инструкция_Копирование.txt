ЗАДАНИЕ 3: МАКРОС РОМБА - ИНСТРУКЦИЯ ПО КОПИРОВАНИЮ
═══════════════════════════════════════════════════════════════

ШАГ 1: Создайте таблицу в Excel
────────────────────────────────
Скопируйте следующее в Excel (начиная с ячейки A1):

┌──────────────┬──────────┬──────────┐
│ Параметр     │ Значение │ Единица │
├──────────────┼──────────┼──────────┤
│ Диагональ 1  │ 10       │          │
│ Диагональ 2  │ 8        │          │
│ Площадь      │          │ ед²      │
│ Угол α       │          │ °        │
│ Угол β       │          │ °        │
│ Сторона      │          │ ед       │
│ Периметр     │          │ ед       │
└──────────────┴──────────┴──────────┘

ИЛИ откройте файл "Задание3_Ромб_Данные.csv"

ШАГ 2: Добавьте макрос
───────────────────────
1. Нажмите Alt+F11 (открыть редактор VBA)
2. Вставка → Модуль
3. Скопируйте код из файла "Задание3_Ромб_Макрос.bas"
4. Сохраните файл как .xlsm

ШАГ 3: Создайте кнопку (ВАЖНО - чтобы не двигалась!)
───────────────────────────────────────────────────────
1. Разработчик → Вставить → Кнопка (Форма)
   (ПЕРВАЯ кнопка в разделе "Элементы управления формы")
2. Нарисуйте кнопку на листе (зажмите мышь и растяните)
3. Автоматически откроется окно "Назначить макрос"
4. Выберите макрос "VychislitParametriRomb" → OK
5. Правый клик по кнопке → "Формат объекта"
6. Вкладка "Свойства":
   - Выберите "Не перемещать и не изменять размеры"
   - ИЛИ "Перемещать и изменять размеры вместе с ячейками"
7. Нажмите OK

Готово! Теперь кнопка закреплена и работает.

ПОДРОБНАЯ ИНСТРУКЦИЯ: см. файл "Задание3_Как_Создать_Кнопку.txt"
