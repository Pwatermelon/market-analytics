ЗАДАНИЕ 8: ОПРЕДЕЛЕНИЕ КОРНЕЙ - ИНСТРУКЦИЯ ПО КОПИРОВАНИЮ
═══════════════════════════════════════════════════════════════

ШАГ 1: Создайте таблицу параметров
───────────────────────────────────
Скопируйте следующее в Excel (начиная с ячейки A1):

┌──────────┬──────────┐
│ Параметр │ Значение │
├──────────┼──────────┤
│ x1       │ 0.1      │
│ x2       │ 10       │
│ A        │ 1        │
│ B        │ -5       │
└──────────┴──────────┘

ИЛИ откройте файл "Задание8_Корни_Данные.csv"

ШАГ 2: Добавьте макрос
───────────────────────
1. Нажмите Alt+F11 (открыть редактор VBA)
2. Вставка → Модуль
3. Скопируйте код из файла "Задание8_ОпределениеКорней.bas"
4. Сохраните файл как .xlsm

ШАГ 3: Запустите макрос
────────────────────────
1. Нажмите Alt+F8
2. Выберите макрос "AnalizKorneyFunktsii"
3. Нажмите "Выполнить"
4. Результаты появятся в столбцах D-J:
   - D1-D5: Параметры функции
   - D7-E7: Наличие корня
   - D10-G10: Заголовки таблицы корней
   - D11-G...: Найденные корни
   - I1-J...: Таблица значений для графика

ШАГ 4: Постройте график
─────────────────────────
1. Выделите данные в столбцах I и J (начиная со строки 2)
2. Вставка → График → График с маркерами
3. Готово!
